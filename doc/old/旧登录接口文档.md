**登录注册模块接口文档**

**1. 用户注册接口**

**接口说明**

用于学生或教师的账号注册

**基本信息**

请求地址：/api/auth/register

请求方法：POST

请求头：Content-Type: application/json

**请求参数**

  ------------ ------------ ------------ -------------------------- -------------------
  参数名称     类型         是否必填     说明                       示例值

  username     string       是           用户名，用于登录           \"zhangsan\"

  password     string       是           密码（建议前端加密传输）   \"encryptedPwd\"

  email        string       否           邮箱地址                   \"user@mail.com\"

  phone        string       否           手机号                     \"13800138000\"

  role         string       是           用户角色：student 或       \"student\"
                                         teacher                    
  ------------ ------------ ------------ -------------------------- -------------------

**请求示例**

  --------------------------------------------------------------
  JSON\
  {\
  \"username\": \"zhangsan\",\
  \"password\": \"encryptedPwd\",\
  \"email\": \"user@mail.com\",\
  \"phone\": \"13800138000\",\
  \"role\": \"student\"\
  }

  --------------------------------------------------------------

**响应参数**

  -------------------- -------------------- ----------------------
  参数名称             类型                 说明

  code                 number               状态码

  message              string               返回信息

  data                 object               返回数据(不返回数据)
  -------------------- -------------------- ----------------------

**响应示例 - 成功**

  --------------------------------------------------------------
  JSON\
  {\
  \"code\": 200,\
  \"message\": \"注册成功\",\
  \"data\": null\
  }

  --------------------------------------------------------------

**响应示例 - 失败**

  --------------------------------------------------------------
  JSON\
  {\
  \"code\": 400,\
  \"message\": \"用户名已存在\",\
  \"data\": null\
  }

  --------------------------------------------------------------

**2. 用户登录接口**

**接口说明**

用于用户登录系统，返回登录凭证

**基本信息**

请求地址：/api/auth/login

请求方法：POST

请求头：Content-Type: application/json

**请求参数**

  ------------ ------------ ------------ ---------------------- ------------------
  参数名称     类型         是否必填     说明                   示例值

  username     string       是           用户名                 \"zhangsan\"

  password     string       是           密码（建议加密传输）   \"encryptedPwd\"
  ------------ ------------ ------------ ---------------------- ------------------

**请求示例**

  --------------------------------------------------------------
  JSON\
  {\
  \"username\": \"zhangsan\",\
  \"password\": \"encryptedPwd\"\
  }

  --------------------------------------------------------------

**响应参数**

  -------------------- -------------------- --------------------
  参数名称             类型                 说明

  code                 number               状态码

  message              string               返回信息

  data                 object               返回数据
  -------------------- -------------------- --------------------

**响应示例 - 成功**

  --------------------------------------------------------------
  JSON\
  {\
  \"code\": 200,\
  \"message\": \"登录成功\",\
  \"data\": {\
  \"token\": \"eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9\...\",\
  \"username\": \"zhangsan\",\
  \"role\": \"student\"\
  }\
  }

  --------------------------------------------------------------

**响应示例 - 失败**

  --------------------------------------------------------------
  JSON\
  {\
  \"code\": 401,\
  \"message\": \"用户名或密码错误\",\
  \"data\": null\
  }

  --------------------------------------------------------------

**3. 退出登录接口**

**接口说明**

用于用户退出登录，清除登录状态

**基本信息**

请求地址：/api/auth/logout

请求方法：POST

请求头：Authorization: Bearer {token}

**请求参数**

无请求体参数，通过请求头传递token

**响应参数**

  -------------------- -------------------- --------------------
  参数名称             类型                 说明

  code                 number               状态码

  message              string               返回信息

  data                 object               返回数据
  -------------------- -------------------- --------------------

**响应示例 - 成功**

  --------------------------------------------------------------
  JSON\
  {\
  \"code\": 200,\
  \"message\": \"退出成功\",\
  \"data\": null\
  }

  --------------------------------------------------------------

**状态码说明**

200：请求成功

400：请求参数错误

401：认证失败

500：服务器内部错误

**注意事项**

密码建议在前端进行加密处理后再传输

token需要在后续请求的Authorization头中携带
